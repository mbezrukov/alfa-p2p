[[Overview]]
== Описание
Сервис предназначен для работы с объектом Cards.

=== Параметры объекта Cards
|===
| Параметр | Тип | Описание

| id
| String
| ID карты

| direction
| String
| Карта списания или зачисления (sender - карты, с которых переводили, recipient - карты, на которые переводили)

| color
| String
| Код цвета карты

| name
| String
| Название карты

| active
| String
| Активная карта? (true/false)

| alfabank
| String
| Карта Альфа-Банка? (true/false)

| number
| String
| Номер карты

| expirationYear
| Integer
| Год окончания срока действия карты

| expirationMonth
| Integer
| Месяц окончания срока действия карты

| lastTransaction
| Date
| Дата последнего перевода

| paymentSystem
| String
| Платежная система

3+| Объект Account

| balance
| Integer
| Текущий баланс

3+| Объект Currency

| code
| String
| Код валюты

| description
| String
| Название валюты

3+| Объект Bank

| id
| String
| ID банка

| name 
| String
| Кириллическое название банка

| latinName
| String
| Латинское название банка

| countryCode
| String
| Код страны

| color
| String
| Цвет банка

|===

[[overview-http-verbs]]
== Список методов

|===
| Метод | Описание

| `GET /cards`
| Получение списка карт клиента с параметрами

| `GET /cards/123`
| Получение параметров заданной карты

| `POST /cards`
| Сохранение новой карты в системе

| `PATCH /cards/123`
| Редактирование параметров карты в системе

|===
