Сервис предназначен для работы с объектом Cards.

=== JSON schema

https://github.com/mbezrukov/alfa-p2p/blob/master/cards-json-schema[JSON schema] 

=== Параметры объекта Cards
|===
| Параметр | Тип | Описание

| id
| String
| ID карты

| direction
| String
| Карта списания или зачисления (sender - карты, с которых переводили, recipient - карты, на которые переводили)

| color
| String
| Код цвета карты

| name
| String
| Название карты

| active
| Boolean
| Активная карта? (true/false)

| alfabank
| Boolean
| Карта Альфа-Банка? (true/false)

| number
| String
| Номер карты

| expirationYear
| Integer
| Год окончания срока действия карты

| expirationMonth
| Integer
| Месяц окончания срока действия карты

| lastTransaction
| Date
| Дата последнего перевода

| paymentSystem
| String
| Платежная система

3+| Объект Account

| balance
| Integer
| Текущий баланс

3+| Объект Currency

| code
| String
| Код валюты

| description
| String
| Название валюты

3+| Объект Bank

| id
| String
| ID банка

| name 
| String
| Кириллическое название банка

| latinName
| String
| Латинское название банка

| countryCode
| String
| Код страны

| color
| String
| Цвет банка

|===

{
  "id": "1",

  "direction": "sender",
  "active": true,
  "alfabank": false,
  "number": "5154 **** 4025",
  "expirationYear": "15",
  "expirationMonth": "07",
  "lastTransaction": "2015-12-15",
  "paymentSystem": "MC",
  "account": {
    "balance": "15250.50",
    "currency": {
      "code": "USD",
      "description": "Доллар США"
      }
    },
  "bank": {
    "id": "12345679",
    "name": "Сбербанк",
    "latinName": "Sberbank",
    "countryCode": "12",
    "color": "#000008"
    }
}
----

[[overview-http-verbs]]
== Список методов

|===
| Метод | Описание

| `GET /cards`
| Получение списка карт клиента с параметрами

| `GET /cards/123`
| Получение параметров заданной карты

| `POST /cards`
| Сохранение новой карты в системе

| `PATCH /cards/123`
| Редактирование параметров карты в системе

|===
